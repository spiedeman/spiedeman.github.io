LeetCode[396-旋转函数]

题目描述

给定一个长度为 n 的整数数组 A。

假设 BkB_kB​k​​ 是数组 A 顺时针旋转 k 个位置后的数组，我们定义 A 的“旋转函数” F 为：

计算F(0),F(1),...,F(n−1)F(0), F(1), ..., F(n-1)F(0),F(1),...,F(n−1)中的最大值。

注意:
可以认为 n 的值小于 10510^510​5​​。

题目分析

F(k)的递推关系为F(k+1) = F(k) + sum(A) - n * A[-k-1]。

Code

def maxRotateFunction(self, A):
    s = sum(A)
    n = len(A)
    f = sum(i * A[i] for i in range(n))
    ans = f
    for i in range(1, n):
        f = f + s - n * A[-i]
        ans = max(ans, f)
    return ans
