数学 & Tex

数学 & Tex

GitBook 通过插件支持数学公式和Tex。当前最常用的两个插件为：mathjax和katex。

启用数学插件

mathjax的支持最全，但是katex的渲染速度更快。

{
    "plugins":["katex"]
}

添加数学公式

这里是行内公式： {% math %}a \ne 0{% endmath %}

这里是块公式
{% math %}
a \ne 0
{% endmath %}
