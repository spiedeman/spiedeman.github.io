配置

配置

所有配置都以JSON格式存储在 book.json 文件中。

字段

gitbook

{"gitbook": ">=2.0.0"}

此选项用来限制生成本书的 GitBook 的最低版本。

description

{"description": "这是一本很棒的书！"}

此选项定义了书本的描述，默认从 README（第一段）中提取。

language

{"language": "zh"}

此选项定义书本语言，默认值是 en。

direction

{"direction": "rtl"}

此选项用来重新设置文字方向。

styles

此选项用来自定义书本的样式。

例子：

{
    "styles":{
        "website":"styles/website.css",
        "ebook":"styles/ebook.css",
        "pdf":"styles/pdf.css",
        "mobi":"styles/monbi.css",
        "epub":"styles/epub.css"
    }
}

plugins

{"plugins":["katex"]}

此选项给出书本所用的插件列表。

pluginsConfig

{
    "plugins":["myPlugin"],
    "pluginsConfig":{
        "myPlugin":{
            "message":"Hello World"
        }
    }
}

structure

这个选项用于覆盖 GitBook 使用的路径。

例如使用 INTRO.md 代替 README.md：

{
    "structure":{
        "readme":"INTRO.md"
    }
}

variables

{
    "variables":{
        "myTest": "Hello World"
    }
}

此选项用来定义在 模版中使用的变量。
